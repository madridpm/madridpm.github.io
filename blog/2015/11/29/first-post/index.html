Statocles & GitHub Pages

Tags: statocles github

Posted on 2015-11-29

Hemos cambiado la web para utilizar un generador estático: Statocles y usar GitHub Pages para no tener que preocuparnos del alojamiento. También hemos creado la organización madridpm en GitHub para gestionar la web y los futuros proyectos en los que trabajemos juntos ;)

Cualquiera puede contribuir a mejorar y extender la web. Es muy sencillo:

  • Hacer un fork del repositorio
  • Clonar tu fork
  • Instalar Statocles
  • Hacer los cambios que creas convenientes
  • Commit & Push de tus cambios
  • Y por último: Pull Request
